# Data Base Manager

---

Simple Graphic User Interface Powered By Python & PySide6

Format Support

> SQLITE3
>
> CSV

Free & Open Source

Create With **Python** Version = 3.9

### Packages

>**Console:**
>
>```
>pip install PySide6			# Gui Core
>pip install cx_Freeze		# Compiler if you need compile
>```
>
>

### Compile 

>**Console:**
>
>```
>python setup.py build
>```

### Run With Python

>**Console:**
>
>```
>python main.py
>```
>
>
